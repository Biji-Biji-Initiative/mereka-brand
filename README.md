# Mereka Brand Guidelines

Central repository for Mereka branding assets and guidelines. Use this as a reference for all Mereka projects to ensure consistent design.

## Structure

```
mereka-brand/
├── logos/          # Logo files (SVG, PNG)
├── colors/         # Color palettes and swatches
├── fonts/          # Typography files and guidelines
├── icons/          # Icon sets
└── templates/      # Design templates
```

## Usage

Reference this repo in your projects:

- Direct asset downloads
